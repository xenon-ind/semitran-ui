# SemiTran UI

**SemiTran** (semi-transparent) is UI library for React written on TypeScript by [Xenon](https://github.com/xenon-ind).


## Description

### Features
- animations: all elements have beautiful customizable animations of clicking, pointing, etc.
- translucent style: containers, secondary buttons, and other UI elements are translucent, as in the graphical interfaces of modern mobile operating systems.
- support for SVG and font icons
- classic but beautiful style
- the possibility of flexible configuration
- a simple API for manually overriding CSS
